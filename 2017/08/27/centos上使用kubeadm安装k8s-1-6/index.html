centos上使用kubeadm安装k8s 1.6

发表于 2017-08-27

去年有一个项目用上了k8s(kubernetes简称)，个人对此非常感兴趣，想搭建集群的，结果等到了现在，不过好歹也是成功搭上了，下面开始教程。

本文将从以下几个部分来介绍安装

  1. 前期准备，主要包括系统的配置和一些软件的安装
  2. k8s安装
  3. k8s配置

说明

  1. docker版本：1.12
  2. k8s版本：1.6.7
  3. 工作机（在这台机器上连接到k8s集群节点进行配置，并且提供代理给各个节点使用）
    ip：192.168.28.99
    系统：windows10
  4. 集群
    系统：都是centos7.3
    主节点，ip：192.168.28.10，hostname: master
    子节点一，ip：192.168.28.11，hostname: node1
    子节点二，ip：192.168.28.12，hostname: node2

安装k8s 1.6.7

使用yum install kubelet-1.6.7-0.x86_64 kubeadm-1.6.7-0.x86_64 kubectl-1.6.7-0.x86_64安装。无所误安装完成即可。

初始化集群

（1）所有节点上执行systemctl enable kubelet，用于开启kubelet服务。
（2）使用
